Bookstore PHP App
=================

This is a PHP app that will enable a hypothetical bookstore manage inventory and sales. Key features include:
+ frontend for customers
+ cart functionality
+ backend for admin users and inventory management

### Technologies Uses
- Linux Box
- Nginx
- PHP
- MySQL
